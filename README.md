## Usefull commands
### Local web server
`symfony server:start -d`

`symfony server:log`

`symfony open:local`

### Maker Bundle
`symfony console make:controller ExampleController`

### Docker Compose
`docker-compose up -d`

`docker-compose ps`

`docker-compose logs`
